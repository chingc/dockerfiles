# teamcity-server

Powerful continuous integration out of the box.


## Description

This `Dockerfile` will build an image that contains a TeamCity Continuous Integration Server.


## Build

`$ docker build -t teamcity-server .`


## Run

`$ docker run --rm -it teamcity-server`
